# libWRP-SQLite3

C++ cross platform key-code library

## Support

- (Supported)
    - GNU/Linux
    - Mac/OSX
- (Comming soon)
    - Microsoft/Windows

## Library

The library is in the "./include" directory.

### Install

to system: install to /usr/local

    sudo ./install.sh

to the other:

    PREFIX=~/opt ./install.sh

## Example

- [key](example/key.cxx)

    cd example
    mkdir build
    cd build
    cmake -G Ninja ..
    ninja
    ./key

## Requirement

* C++11
